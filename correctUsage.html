How to use fonts correctly?

1. Font Size
The eye is naturally drawn to large or dominant elements, so choose the largest font size for your title, followed by your subtitle, then your body text.

2. Use at most 3 fonts, all of which should work together
Having more than 3 fonts can start confusing the viewer as the variety of different fonts may make the page hard to look at and all over the place.

3. Font Color
Make sure the backgroud color doesn't make the text impossible to read. To avoid this, make the font color easiliy readble based on the background.

4. Create Readability
Font readability is especially important for longer sections of text. Try to avoid using elaborate (script) fonts or upper case text, as these options can strain the eye. Save these for titles or headings

5. Make the font align with the purpose
For example, if it is a formal webpage use a formal font like Times New Roman or Arial as they are known to be used in formal texts. However, if the webpage is about a fantasy story, then use the curvy and fancy font that creates the feeling of a fantasy story/

6. Be Creative
Have fun with different fonts and finding which fonts you like working with best
